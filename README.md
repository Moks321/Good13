# Good13
Good 13 is a simple, single-player card game that uses a standard deck of 52 playing cards developed using Java.
The object of the game is to remove all 52 cards from th eplaying area by matching them in one of 2 ways as given below:
    - by removing pair of cards for which the combined rank value is 13 regardless of their suit
    - by removing individual Kings which are the only cards having a value of 13.
